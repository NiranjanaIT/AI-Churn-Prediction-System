# OTT Churn Prediction AI

A hackathon-ready, end-to-end full-stack application that uses AI to predict user churn for an OTT subscription platform and recommends personalized retention actions.

## 🚀 Hackathon Pitch
Customer retention is 5x cheaper than acquisition. This "OTT Agent" uses real-time behavioral data (Watch Hours, Payment History, Activity Recency) and a custom TensorFlow.js Neural Network to identify at-risk users *before* they cancel. It's not just a dashboard; it's an actionable retention engine.

## 🛠 Tech Stack
- **Frontend**: React.js, Tailwind CSS, Chart.js, Lucide Icons
- **Backend**: Node.js, Express.js
- **Database**: MongoDB Atlas
- **AI/ML**: TensorFlow.js (Sequential Neural Network)

## 🧠 AI Explanation
The core logic resides in `backend/ml/churnModel.js`.
1. **Model**: A 3-layer sequential neural network.
2. **Features**: 
   - `lastActiveDays` (Normalized 0-1)
   - `watchHours` (Normalized 0-1)
   - `paymentDelay` (Normalized 0-1)
3. **Training**: The model trains on-the-fly (or via pre-trained weights) using synthetic ground-truth patterns derived from industry churn metrics.
4. **Prediction**: Returns a probability score (0-1) which is mapped to LOW, MEDIUM, or HIGH risk.

## 🏃‍♂️ How to Run

### Prerequisites
- Node.js installed
- MongoDB Atlas URI

### 1. Backend Setup
```bash
cd backend
npm install
# Update .env with your MONGODB_URI if needed
npm run seed  # Optional: Populates dummy users
node server.js
```
Server runs on port 5000.

### 2. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
Frontend runs on `http://localhost:5173`.

### 3. Demo Flow
1. **Login**: Use `admin@ott.com` / `admin123`.
2. **Dashboard**: View high-level stats and recent AI activity.
3. **Users**: Browse user base and click "Predict" on any user.
4. **Prediction**: See real-time inference result.
5. **Action**: View the personalized retention strategy generated by the system.
